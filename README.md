## Запуск проекта

### Список предварительных действий:

- установить Docker
- установить NodeJs

### Выполнить скрипты:

1. npm run expand
2. npm run up
3. npm run start:server
4. npm run start:client

### Остановить сервисы

1. npm run down
2. Перейти в запущенный терминал клиентского приложения
3. Выполнить сочетание клавишь Ctrl+C
